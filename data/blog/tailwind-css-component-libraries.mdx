**Tailwind css** es una herramienta de css que nos permite crear componentes de forma rápida y sencilla. Tailwind ha sido muy bien aceptado en la comunidad front-end, gracias a la agilidad que nos brinda a la hora de crear aplicaciones web y prototipos a una gran velocidad.

En este post te comparto una colección de librerías de componentes open-source para Tailwind css.

## DaisyUI

[daisyUI](https://daisyui.com/)
Es una librería css gratuita y open source que contiene una colección de **48 componentes**, cada componente con previsualización y código de ejemplo en `HTML` y `JSX`. Extiende tailwind usando plugin y una configuración para elegir que integrar en tu proyecto.

Lo recomiendo por lo sencillo que es de usar y la gran cantidad de componentes que contiene, además nos brindan un [generador de themes](https://daisyui.com/theme-generator/) para personalizar los colores de los componentes.
